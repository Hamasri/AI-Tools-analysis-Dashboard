# AI-Tools-analysis-Dashboard
This dashboard provides an analysis of popular AI tools based on users, ratings, pricing type, industry usage, and monthly growth. It helps identify the most used AI tools, compare free vs paid tools, and understand industry-wise adoption trends.
KPI Cards Questions:
  Total AI Tools
  Total Users
  Average Rating
  Average Monthly Growth
  Average Productivity Score
  Paid vs Free Tool Count
  Highest Used AI Tool
  Fastest Growing AI Tool
